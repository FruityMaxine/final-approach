# FINAL APPROACH — 硬核浏览器飞行模拟器

> 第一视角客机进近着陆模拟器。纯原生 `canvas` / WebAudio / 零依赖。
> 目标：**高拟真 · 全功能 · 游戏性** —— 像真驾驶舱那样可扩展出数十种系统面板，一切功能 / 故障 / 难度 / 环境皆可开关。

手机 / iPad / 电脑三屏自适应，打开即玩，无需安装。

## 在线试玩

挂载于 Vega Arcade 游戏盒（`final-approach` 卡 → 全屏直跳）。本仓为独立开源源码。

## 玩法

坐在窄体客机左座，沿 **3° 下滑道**对准 27 号跑道：守住下滑道（PAPI 两白两红）、对准中线、速度锁 Vref 140kt，入口柔和拉平，接地后反推 + 刹车停住。下降率越小，讲评评分越高。

**操控**：摇杆（俯仰 / 横滚）· 推力杆 · 襟翼 · 方向舵脚蹬 · GEAR/SPLR/REV/BRK/TO-GA/EMMA。
键盘：`W/S` 俯仰，`A/D` 横滚，`Z/C` 方向舵，`Shift/Ctrl` 油门，`F/V` 襟翼，`G` 起落架，`B` 扰流板，`空格` 刹车/反推，`Enter` 复飞，`E` EMMA。

## EMMA AI 副驾（分轴接管）

点亮 EMMA 把控制权交给 AI 副驾。**分轴接管**：你操作单一通道（只动摇杆 / 只动脚舵 / 只动油门）时，EMMA 只交还你正在操作的那根轴，其余轴继续飞——不会因为你碰一下就全撒手。还有 **assist 辅助驾驶档**：只给 HUD 飞行指引十字、不抢杆。

## 难度与配置

三档难度（新手 / 真实 / 硬核对标微软模拟飞行）影响风 / 湍流 / 风切变 / 容差 / 判定。可配起始位置（短/长五边、高位切入）、风向风速、时段（黄昏/正午/夜间）、自由飞模式。全部 `localStorage` 持久化。

## 架构（模块化，纯静态多文件）

| 模块 | 职责 |
|---|---|
| `web/sys.js` | **中央 SYS 开关注册表** —— 功能 / 故障 / 环境 / 面板统一开关（"一切皆可开关"基石） |
| `web/config.js` | 难度预设 / 起始位置 / 天气 / 持久化 |
| `web/render.js` | 3D 投影 + 外景世界（机场 / 地形网格 / 城市天际线 / 三时段光照）+ PFD/HUD |
| `web/emma.js` | EMMA AI 副驾（分轴接管状态机） |
| `web/game.js` | 飞行物理 / 输入 / 讲评 / 设备三屏 / 主循环装配 |
| `web/game.css` | 全部样式 + 三屏设备布局层 |

普通 `<script>` 按序加载共享全局作用域，无打包器、无构建步骤。

## 路线图（北极星：高拟真 · 全功能 · 游戏性）

- **面板框架**：面板注册表 + 驾驶舱式选择器，可无限扩展数十种面板（引擎 / 燃油 / 液压 / 电气 / 增压 / APU / 起落架 / 防冰 / 灯光 / 无线电 / 自动驾驶 MCP / …）
- **多发引擎系统**：可配 2/4 发，独立 N1/N2/EGT/燃油流量 / 点火 / 灭火 / 启动机
- **燃油系统**：左中右油箱 / 燃油泵 / 交输活门 / 耗油 / 燃油饥饿熄火
- **MCDU 飞控电脑**（旗舰件）：屏幕四周一圈实体键（1L–6L / 1R–6R 行选键 LSK + 字母数字键盘 + 功能键），多页 CDU
- **故障系统**：引擎失效/起火、液压、电气、燃油泄漏、爆胎、仪表失效（手动触发 + 概率 + 连锁影响）
- **高级现象**：飞控电脑 FBW 法则、仪表飞行 IMC、空间迷向、风切变、尾流、结冰
- **程序化世界**：算法生成地形 / 城市连续铺展、渲染优化
- **新手分步交互引导教学**
